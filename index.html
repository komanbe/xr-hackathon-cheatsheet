02 ── World API (World Labs) ディープダイブ
OVERVIEW

World API ── AI駆動の3D世界生成プラットフォーム

Fei-Fei Li率いるWorld Labsが開発。マルチモーダルワールドモデル「Marble」をAPI経由で利用し、テキスト・画像・動画から探索可能な3D世界を自動生成する。

入力モダリティ

  • テキストプロンプト「廃墟の教会、朝霧」のような自然言語で世界を生成
  • 単一画像1枚の写真からシーン全体を3D再構築
  • 複数画像複数アングルの写真でより精密な世界を生成
  • 360°パノラマパノラマ画像から没入型環境を構築
  • 動画映像から空間レイアウト・奥行き・照明を推定

出力・エクスポート

  • WebレンダリングWebGL対応。ブラウザ上で即座に探索可能
  • glTF / USD標準3Dフォーマットでゲームエンジンに取り込み
  • VR体験Quest 3 / Vision Pro で直接表示可能
  • メッシュエクスポート3Dプリント・CAD・モデリングソフト向け
  • DCC統合Blender / Maya / 3ds Max に直接インポート
CORE FEATURES

Marble ワールドモデルの機能

  • 生成時間 約30秒テキスト/画像/3D構造からパノラマ生成まで
  • Chisel Editor壁・部屋・地形などの粗い3Dレイアウトを描き、AIがディテールを補完する実験的エディタ
  • Expansion（拡張）既存世界のルール・スタイルに基づいてさらに広い3D空間を自動生成
  • Composer Mode異なるスタイルの生成済み世界をつなぎ合わせて巨大空間を構築
  • AI-native編集生成後もテキスト指示で照明・テクスチャ・構造を変更可能
  • ハイブリッド3Dエディタ手動ブロックアウト → AIビジュアライゼーションの2段階ワークフロー

03 ── CesiumJS ディープダイブ
OVERVIEW

CesiumJS ── 地球規模の3D地理空間エンジン

OGC標準「3D Tiles」の策定者が開発するオープンソースJavaScriptライブラリ。WebGLベースの3Dデジタル地球儀で、衛星画像・地形・都市3Dモデル・地理データをリアルタイム表示。

コア機能

  • 3D地球儀衛星画像+地形データの高精度レンダリング
  • 3D Tiles (OGC標準)都市3Dモデルを地球規模でストリーミング配信
  • Gaussian Splats対応2025年から3DGSデータをCesium上に表示可能に
  • 時系列データTime-Dynamic 3D Tilesで時間変化を可視化
  • パノラマ画像Google Maps 360°パノラマをCesium内に統合
  • Imagery Draping任意の画像レイヤーを3D Tilesの上に重ねられる

エコシステム

  • Cesium ionクラウドタイリング・ホスティング・アセットの一元管理
  • Cesium for UnityUnity内に3D地球儀を直接表示。WebGPU対応
  • Cesium for UnrealUE5のNanite/Lumenと地理空間データの統合
  • Three.js統合three-loader-3dtilesで3D TilesをThree.jsに
  • Google Photorealistic 3D Tiles現実都市をそのままCesiumに取り込み
  • AI SearchSandcastleでのセマンティック検索（2025末GA）

撮影のコツ

  • Photo Scan対象の周りを一周しながら50〜150枚撮影。重なり70%以上が理想
  • 3DGS Scan動画で対象をゆっくり一周。30秒〜2分。手ブレ厳禁
  • LiDAR ScaniPhone Pro/iPad Proで空間全体をスキャン。広い室内に最適
  • 照明均一な照明が理想。強い影・逆光は避ける
  • 苦手な対象鏡面・透明物体・極端に薄い物体・動く被写体

3DGS to Mesh 変換

  • 業界初の変換機能KIRI Engineが3DGS→メッシュ変換を初めて実装
  • 有効化方法アップロード時に「3DGS to Mesh」オプションをONにする
  • UV・テクスチャ自動生成3DGSの色/輝度データからUVマップとテクスチャファイルを生成
  • 後処理可能変換後はリトポロジー・スカルプト・UV編集・リギングが可能
  • ゲームエンジン直行OBJ/glTF変換後はUnity/UE/Godotに直接インポート

06 ── Arrival Space ディープダイブ
OVERVIEW

Arrival.Space ── 3DGSをWebで公開・編集・体験する最速プラットフォーム

3D Gaussian Splattingに特化したソーシャルWebプラットフォーム。PLY/SPZファイルをアップロードするだけで、ブラウザ上で閲覧・編集・共有・VR体験ができる。モバイル/PC/VRヘッドセット全対応。

コア機能

  • 3DGSホスティングPLY/SPZをアップロード → URL発行 → 即共有
  • SuperSplat統合プラットフォーム内で3DGSを直接編集・クリーンアップ
  • Spherical Harmonics最大3バンドのSH対応で高品質ビジュアル
  • 自動衝突判定アバターがSplatシーン内をリアルに歩ける
  • Vibes (プロンプト演出)テキストプロンプトでシーンにエフェクトやロジックを追加
  • SPZファイル完全対応アップロード・編集・変換すべて対応

技術スタック

  • PlayCanvas EngineWebGL/WebGPU/WebXR/glTF対応のWebグラフィックスエンジン
  • HDR対応ハイダイナミックレンジレンダリング
  • Splat Data Protectionクリエイターの3DGSデータを保護する仕組み
  • splat-transform (OSS)GitHub公開のCLIツール。3DGSフォーマット変換
  • spz-js (OSS)GitHub公開。SPZ↔PLYをJSで変換
  • マルチデバイスモバイル / PC / VRヘッドセットすべて対応
WORKFLOW

Arrival Space 利用フロー

  • 1. アカウント作成arrival.space で自分のSpaceを作成（無料）
  • 2. 3DGSデータをアップロードPLY/SPZファイルをドラッグ&ドロップ。複数ファイル対応
  • 3. SuperSplatで編集不要な部分のクロップ・ノイズ除去をブラウザ内で完結
  • 4. Vibesで演出追加テキストプロンプトでエフェクト・ロジックを生成
  • 5. URLで共有arrival.space/あなたのID でアクセス可能。VRヘッドセットからも
KIRI / Scaniverse → Arrival Space: スキャンアプリでPLY/SPZをエクスポート → Arrival Spaceにアップロード → 数分で3DGS展示空間が完成。コード不要。
